# gather-app
Activity-based micro-hangouts for adults. Groups of 4–8 free forever. Premium up to 37 people. PWA · Stripe · Real-time chat.
Gather — Find your people. Not just a match.
A mobile-first PWA for activity-based friend-making. No swiping, no algorithms, no awkward one-on-ones. Just small groups doing real things together — hiking, board games, coffee, art. Groups of 4–8 are always free. Bigger groups unlock Premium tiers up to 37 people.
Built with vanilla HTML/CSS/JS · Stripe payments · Supabase backend · Installable on iOS and Android.
